# Bilder für die Website

Speichere deine Bilder in diesem Ordner mit diesen Dateinamen:

## Deine Bilder (die du hast):

1. **salon-2.jpg** → Hero Bild + "Warum dieser Salon" Bereich
   
2. **0_1.jpg** → Haarbilder in Galerie (1)
3. **0_2.jpg** → Haarbilder in Galerie (2)
4. **0_3.jpg** → Haarbilder in Galerie (3)

5. **0_5.jpg** → Gebäude/Außenaufnahme in Galerie

6. **oxana-logo.jpg** oder **oxana-logo.png** → Logo mit Schneidwerkzeugen
   (wird auch in der Galerie angezeigt)

---

## Dateistruktur:

```
public/images/
  ├── salon-2.jpg      (Dein Salon Bild)
  ├── 0_1.jpg          (Haarbilder)
  ├── 0_2.jpg
  ├── 0_3.jpg
  ├── 0_5.jpg          (Gebäude)
  ├── oxana-logo.png   (Logo - optional .png oder .jpg)
  └── README.md        (diese Datei)
```

---

## So funktioniert's:

1. Kopiere deine Bilder in diesen Ordner **public/images/**
2. Benenne sie genau wie oben (wichtig: Groß-/Kleinschreibung und Dateiendung!)
3. Fertig! Die Website lädt die Bilder automatisch.

---

## Bildformate:

- JPG/JPEG für Fotos (schnell, komprimiert)
- PNG für das Logo (unterstützt Transparenz)

## Bildgröße:

- **salon-2.jpg**: mindestens 600x500px
- **Haarbilder (0_1, 0_2, 0_3)**: mindestens 500x500px
- **0_5.jpg (Gebäude)**: mindestens 500x500px
- **oxana-logo**: mindestens 200x200px
